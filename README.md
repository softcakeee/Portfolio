### 졸업작품: 금속탐지기 개발 1 (2024)
-  대학교 4학년 1학기 졸업 작품 입니다.

-  참여 인원 수
  - 1명

- 개발 기간
  - 2024.03 ~ 2024.06
 
-  프로젝트 내용
    - 매설물 중 철근 더 나아가 도체를 탐지하는 GPR 같은 역할을 하는 금속탐지기를 개발하고자 함.
    - 코일 전류를 펄스를 발생시켜 자기장을 발생시킬 때 생기는 자기장에 도체가 있을 시 전류의 흐름이 크게 변화하게 되는 것을 이용하고자 함.
    - 탐지 시 도체의 유무를 시각화하고자 함.
    - 도체를 다중 탐지할 시 도체 사이에 거리를 탐지하기 위하여 금속탐지기의 이동거리를 얻기 위하여 홀센서를 이용함.

     
 -  수행 내용
    - 프로그래밍 및 회로 설계/제작
    - 사용언어 C
    - github : https://github.com/softcakeee/Metal_Detector_1
    - 펌웨어 개발 (아두이노 나노)
        - 코일을 활용한 도체 탐지 알고리즘 개발.
        - 무선송수신 모듈을 이용한 무선 통신 알고리즘 개발.
        - 데이터 값 시각화 알고리즘 구현.
        - 홀센서를 이용한 이동거리 측정 알고리즘 개발.
