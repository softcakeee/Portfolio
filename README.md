### 졸업작품: 금속탐지기 개발 1 (2024)
-  대학교 4학년 1학기 졸업 작품 입니다.

-  참여 인원 수
  - 1명

- 개발 기간
  - 2024.03 ~ 2024.06
 
-  프로젝트 내용
    - 매설물 중 철근 더 나아가 도체를 탐지하는 GPR 같은 역할을 하는 금속탐지기를 개발하고자 함.
    - 코일 전류를 펄스를 발생시켜 자기장을 발생시킬 때 생기는 자기장에 도체가 있을 시 전류의 흐름이 크게 변화하게 되는 것을 이용하고자 함.
    - 탐지 시 도체의 유무를 시각화하고자 함.
    - 도체를 다중 탐지할 시 도체 사이에 거리를 탐지하기 위하여 금속탐지기의 이동거리를 얻기 위하여 홀센서를 이용함.

     
 -  수행 내용
    - 프로그래밍 및 회로 설계/제작
    - 사용언어 Arduino(C/C++)
    - github : https://github.com/softcakeee/Metal_Detector_1
    - 펌웨어 개발 (아두이노 나노)
        - 코일을 활용한 회로에서의 전류를 ADC를 사용하여 도체 탐지 알고리즘 개발.
        - 홀센서를 이용한 이동거리 측정 알고리즘 개발.
        - SPI를 사용하는 무선송수신 모듈을 이용한 무선 통신 알고리즘 개발.
        - PLX-DAQ 활용한 데이터 값 시각화 알고리즘 구현.

 -  결과 및 느낀점
    - 코일을 만들때 감은 횟수에따라 감도가 달라져서 최적화하는데 어려움을 겪었고 이때 센서의 감도를 프로젝트에 맞게 조정하는것도 많은 시간과 노력이 필요하겠다고 생각이 들었음.
    - 금속탐지기의 이동거리를 위하여 홀센서를 사용하였는데 이때 홀센서를 사용하는 아이디어는 교수님께서 내주셨음 홀센서는 자석인식하는 센서라고만 생각하였는데 그걸 이용해서 바퀴수를 계산하는것을 알고 센서의 기본적인 기능만이 아니라 그
     기본적인 기능을 어떤식으로 사용하느냐에따라서 활용도가 달라진다라는 것을 알았음.

### 졸업작품 : 금속탐지기 개발 2 (2024)
-  대학교 4학년 2학기 졸업 작품 입니다.

-  참여 인원 수
  - 1명

- 개발 기간
  - 2024.09 ~ 2024.12
 
-  프로젝트 내용
   - 1학기에 느꼈던 부족했던 부분을 해결하고자 함.
   - 단일 코일 방식의 낮은 해상도를 극복하기 위해 3채널 분할 코일 구조 도입 하고자 함.
   - 각 코일에 도체 탐지 여부를 시각화 하고자 함.
   - 1학기때의 탐지기 형태에 한계를 느낀 부분을 4륜으로 변경하여 해결하고자 함.
   - 탐지를 위해서 인간이 직접 밀어야하는 형태를 해결하고자 함.
 
 -  수행 내용
    - 프로그래밍 및 회로 설계/제작
    - 사용언어 C 및 python
    - github : https://github.com/softcakeee/Metal_Detector_2
    - 로봇 제어 프로그램 개발 (Raspberry PI)
       -  상위 제어 로직 구현.
       -  pygame 라이브러리를 이용한 게임 패드와 연결 후 데이터 처리 알고리즘 구현.
       -  STM32 와 UART 통신 알고리즘 구현.
       -  matplot 라이브러리를 이용한 데이터 값 그래프 시각화 알고리즘 구현.
         
    - 펌웨어 개발 (STM32F103)
        - 여러개의 코일을 활용한 도체 탐지 알고리즘 개발.
        - Raspberry pi 와 UART 통신 알고리즘 구현.
        - UART 통신을 활용한 조이스틱 모터제어 알고리즘 개발.

 -  결과 및 느낀점
    -   학교에서의 마지막 프로젝트라 생각해서 지금까지 학교에서 경험했던 부분들을 한 프로젝트에 담고자 하였음
    -   RPI 와 MCU 사이에서의 데이터 교환 부분을 2학년때의 부족했던 부분을 해결하였고 실력이 늘었구나 라고 생각하는 계기가 되었음.
    -   지금까지 사용했던 MCU 는 동아리에서 따로 배웠던거나 학교에서 배웠던 것을 사용했지만 처음으로 새로운 MCU 를 사용하면서 처음 개발을 할때 개발환경 구축과 사용법 을 독학하는 경험을 하게되었음
